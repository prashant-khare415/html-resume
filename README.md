# 💼 Personal Portfolio Resume Website
A responsive and clean resume website built using HTML. This project showcases my personal profile, education, skills, projects, and contact information in a simple and professional web layout.

## 🚀 Features
- Professional resume layout
- About Me section
- Education details
- Technical Skills
- Projects section
- Contact Information
- Clean and beginner-friendly HTML structure

## 🛠️ Technologies Used
- HTML5

## ▶️ How to Run
1. Clone this repository.
2. Open `html-resume.html` in any web browser.
